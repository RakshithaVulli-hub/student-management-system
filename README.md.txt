# Student Management System (Python + Pandas)

## Description
A console-based Student Management System built using Python and Pandas.
Supports CSV & Excel storage.

## Features
- Add / Update / Delete student records
- Search by Roll No or Name
- Automatic Excel synchronization
- Deleted records backup

## Technologies
- Python
- Pandas
- CSV / Excel

## How to Run
```bash
pip install -r requirements.txt
python main.py
